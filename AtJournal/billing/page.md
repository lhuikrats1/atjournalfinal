# Billing

_Exported from billing/page.tsx_

## Purpose

Manages subscription plans, payment methods, and invoicing for AtJournal users. Integrates with Stripe for secure transactions.

## Features
- View current plan and usage limits.
- Upgrade / downgrade subscription.
- Add / remove payment cards.
- Download invoices (PDF).
