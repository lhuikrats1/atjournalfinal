# Vault

_Exported from vault/page.tsx_

## Purpose

Provides a directory of all markdown notes stored in the Obsidian vault. Users can click a note to open its full content.

## Features
- Lists every markdown file in the vault with a clickable link.
- Shows a live count of total records.
- Simple UI with a dark theme that matches the AtJournal aesthetic.
