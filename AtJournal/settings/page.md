# Settings

_Exported from settings/page.tsx_

## Purpose

Provides UI for user preferences, theme selection, notification toggles, and account configuration. Settings are stored in the Supabase user profile and persisted across sessions.

## Features
- Theme (light/dark/auto) toggle.
- Email & password management.
- Notification preferences.
- API key management for Obsidian integration.
