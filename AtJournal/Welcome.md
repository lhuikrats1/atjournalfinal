# Welcome to AtJournal Vault

This vault mirrors the AtJournal web application. Each markdown file represents a page or component, enriched with front‑matter and bidirectional links for easy navigation within Obsidian.

## How to use
- Browse the **[[Index]]** for a high‑level map.
- Start at the **[[Home]]** page.
- Follow the **Related** sections at the bottom of each page to jump between connected pages.
- The vault stays in sync with the codebase via the export script.
